# ☕ Cafetería

Sitio web sencillo para una cafetería: muestra el menú y los horarios.
Proyecto de práctica para aprender el flujo de trabajo con Git y GitHub.

## Estructura

```
Cafeteria/
├── index.html   # Página principal
├── styles.css   # Estilos
├── app.js       # Lógica del menú
└── README.md    # Este archivo
```

## Cómo verlo

Abre `index.html` en tu navegador. No requiere instalación.

## Menú

El menú se define en `app.js` y se muestra automáticamente en la página.
